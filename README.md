# 🎙️ Smart Voice Alarm Pro 2026

Una aplicación de alarma futurista diseñada como **PWA (Progressive Web App)**. Permite el control total mediante comandos de voz naturales, optimizada para un uso fluido en dispositivos móviles con estética Neón-Cyberpunk.

## ✨ Características Principales
- **IA de Procesamiento de Voz:** Reconoce fechas específicas (ej. "20 de mayo"), horas en formato AM/PM y alarmas diarias.
- **Reloj de Precisión:** Interfaz centrada con segundero dinámico y efectos de brillo neón.
- **Micro Persistente:** Durante la activación de la alarma, el sistema escucha automáticamente comandos para detenerla.
- **Diseño sin Barras:** Interfaz limpia con scroll invisible para las listas de alarmas.
- **Instalable:** Funciona como una app nativa gracias al Service Worker integrado.

## 🚀 Instalación
1. Sube todos los archivos a un repositorio de GitHub.
2. Ve a **Settings > Pages** y activa el despliegue desde la rama `main`.
3. Abre la URL generada en tu móvil.
4. **IMPORTANTE:** Toca la pantalla al cargar por primera vez para habilitar el motor de audio del navegador.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Flexbox & Grid)
- JavaScript Vanilla (Web Speech API)
- PWA (Service Workers & Manifest)
